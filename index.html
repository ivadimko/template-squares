• Исходные данные в файле json.
    • Load Data - загружает json
    • Draw Squares - рисует квадраты по заданным условиям
    • Clear All - очищает поле
  • После отрисовки квадратов доступны следующие кнопки
    • Enable Border - рисует границу квадратам по заданным условиям
    • Repaint Squares - перекрашивает квадраты по заданным условиям
    • Drop Squares - каждый четвертый квадрат будет с новой строки
    • Add Shadow - каждый второй квадрат получает тень
    • Sort - сортировка квадратов согласно условиям
  • Дополнительные условия
    • Hover Effect - эффекты при наведении согласно условиям
    • Local Storage - работает local storage согласно условиям
